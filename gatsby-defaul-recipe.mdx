# Default Gatsby Setup

This recipe sets up a Gatsby site with all my default plugins and packages. It is going to install/set up:

* TypeScript
* Linaria as a css-in-js solution

Install packages

For TypeScript

<NPMPackage name="gatsby-plugin-typescript" />
<NPMPackage name="@typescript-eslint/parser" dependencyType={"development"} />
<NPMPackage name="@typescript-eslint/eslint-plugin" dependencyType={"development"} />

For linaria

<NPMPackage name="linaria" />
<NPMPackage name="gatsby-plugin-linaria" />
<NPMPackage name="linaria" />

Installing the plugins in your `gatsby-config.js`.
